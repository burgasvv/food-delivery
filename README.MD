
# Сервис доставки еды

##  Версии
* JDK 23
* SpringBoot 3.3.5

## Основной стек зависимостей
* Spring Cloud Gateway
* Spring Security
* Spring Web
* Spring Data JPA
* Springdoc MVC
* Postgresql

## Описание
        Сервис предоставляет возможность регистрации пользователей и сотрудников,
    совершать заказы на еду и напитки, с реализацией корзины с продуктами.
